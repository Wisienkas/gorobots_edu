35% testing = 4900 data points
65% training = 9100 data points

The data was recorded in a bipolar manner from Stoyan performing left and right hand motor imagery.

The input file contains the bipolar recording between channels: C3-F3.
The input data is scaled between -1 and +1.

The target file has the same size as input. There are 5 trials marked where right hand motor imagery was performed. 
During motor imagery, the values in the target set are 1, otherwise they are -1.
